# 🧹 Google Drive Organization

A utility project to maintain a clean, "no nonsense" Windows-style structure on your Google Drive.

## Standard Structure

- `Documents/` — Personal and work docs
- `Photos/` — Images and art (including Wombo art in `Photos/Dream`)
- `Downloads/` — Temporary files
- `Videos/` — Movies and clips
- `Music/` — Songs and albums
- `Archive/` — Older files (School, old sorting folders, etc.)

## Usage

Run the organization script in Termux to clean up existing folders:

```bash
curl -sL https://raw.githubusercontent.com/Mrswami/macrodroid/main/drive-org/organize_drive.sh | bash
```

## Files

| File | Purpose |
|------|---------|
